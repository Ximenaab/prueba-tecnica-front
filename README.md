# Panel Administrativo - Panadería 🥐

Panel de control para gestión de inventario de una panadería. 
Permite administrar productos con operaciones CRUD completas.

## Stack Tecnológico
- React + Vite
- React Router DOM
- Tailwind CSS
- Axios
- SweetAlert2
- MockAPI

## Funcionalidades
- Login simulado con LocalStorage
- Protección de rutas
- Listar productos en tarjetas
- Buscar por nombre o categoría
- Crear, editar y eliminar productos
- Confirmación con SweetAlert2 al eliminar
